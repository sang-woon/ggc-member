# 경기도의회 의원 의정활동 관리시스템

모바일 최적화 의정활동 관리 플랫폼

## Live Demo
- https://sang-woon.github.io/ggc-member/

## 주요 기능
- 대시보드 및 통계
- 의원 정보 관리
- 위치 기반 활동 (블록체인 인증)
- 의안/발언/출석 관리
- 민원 처리 시스템

## 기술 스택
- Vanilla JavaScript (ES6+)
- HTML5, CSS3
- Tailwind CSS
- Chart.js, QRious
- PWA Support

## 시작하기
브라우저에서 index.html 파일을 직접 열거나 웹 서버에서 실행하세요.
